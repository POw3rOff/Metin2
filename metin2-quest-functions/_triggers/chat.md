# chat
Триггер **chat** запускает событие, когда игрок выбирает um determinado пункт в диалоге у определенного NPC.

## Parâmetros
### mob_vnum
Tipo *number*. **Parâmetro obrigatório**. Vnum NPC, с которым необходимо начать диалог.

### message
Tipo *string*. **Parâmetro obrigatório**. Пункт, который игроку необходимо выбрать в диалоге с NPC.

## Notas
Функция [setskin](../global/setskin.md)() с параметром `0` удаляет черные полосы с экрана и заставляет UI снова отображаться. Se после вызова этого триггера не идет функция [say](../global/say.md)(), então черные полосы остаются. В этом случае их надо убирать самостоятельно.
