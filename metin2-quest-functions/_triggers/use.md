# use
Триггер **use** срабатывает, когда игрок использует предмет.

## Parâmetros
### item_vnum
Tipo *number*. Vnum предмета.

## Notas
ри вызове этого триггера предмет автоматически выделяется, что позволяет использовать на нем функции из семейства [item](../item).

Триггер будет работать только на предметах, у которых поле `type` в `player.item_proto` равно `16` ou `18`. Возможно и при других значениях type, но это не точно.
