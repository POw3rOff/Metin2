# party_kill
Триггер **party_kill** срабатывает, когда кто-então внутри группы убивает um monstro ou игрока. Все, что будет происходить после вызова этого триггера, будет относиться только к лидеру группы.

## Parâmetros
### mob_vnum
Tipo *number*. Vnum do monstro.

## Notas
Триггер не будет срабатывать, se um monstro убьет игрок, который находится на другом канале относительно лидера группы. Возможно, будут проблемы, se игроки будут находиться на разных подканалах.

Триггер не сработает, se игрок будет находиться вне группы. Чтобы избежать этого, следует использовать примерно такую конструкцию:

````lua
when 101.kill or 101.party_kill begin
````

Не следует путать этот триггер с конструкцией `when kill with party.is_party() begin`, поскольку эта конструкция срабатывает только в том случае, se игрок убил um monstro, находясь в группе, а этот триггер срабатывает в том случае, se кто-então из участников группы, включая и лидера, убил um monstro.
