# horse.get_name()
A função **horse.get_name** informa имя лошади.

## Valores de retorno
### horse_name
Tipo *string*. Имя лошади. Se функция была вызвана от имени сервера, então é retornado uma string vazia, ou seja `""`. Что é retornado в случае, se у лошади игрока стандартное имя, неизвесно.

## Notas
A função **não** pode ser chamada anonimamente.
