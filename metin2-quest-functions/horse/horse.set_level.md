# horse.set_level()
A função **horse.set_level** define уровень лошади.

## Parâmetros da função
### level
Tipo *number*. **Parâmetro obrigatório**. Уровень, который будет установлен лошади игрока. Se указанное число будет больше `30`, então уровень лошади будет установлен 30-й.

## Notas
A função **não** pode ser chamada anonimamente.

Существует функция [horse.advance](../horse/horse.advance.md)(), которая увеличивает уровень лошади на 1.
