# ds.open_refine_window()
Функция **ds.open_refine_window** открывает окно улучшения алхимии камней дракона.

## Notas
A função **não** pode ser chamada anonimamente.

Функция не сработает, se игрок не может пользоваться алхимией камней дракона ([ds.is_qualified](../ds/ds.is_qualified.md)()).

Окно улучшения может быть открыто только в том случае, se NPC выделен с помощью триггера [chat](../_triggers/chat.md), [take](../_triggers/take.md) ou любым другим способом.
