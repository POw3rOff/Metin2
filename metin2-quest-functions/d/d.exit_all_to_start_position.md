# d.exit_all_to_start_position()
A função **d.exit_all_to_start_position** teletransporta каждого игрока na masmorra на заданые функциями [pc.set_warp_location](../pc/pc.set_warp_location.md)(), [pc.set_warp_location_local](../pc/pc.set_warp_location_local.md)() и [d.set_warp_location](../d/d.set_warp_location.md)(). координаты. Se координаты не были заданы этими функциями, então игроки телепортируются в первые города.

## Notas
A função pode ser chamada anonimamente.

Esta função funciona apenas em masmorras.
