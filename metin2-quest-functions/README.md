# Руководство по квестовым функциям Metin2

<p align="center">
	<img src="https://raw.githubusercontent.com/terron-kun/quest-functions/master/_images/banner.gif" alt="Логотип">
</p>

Данный репозиторий содержит в себе зачастую исчерпывающую документацию по квестовым функциям Metin2. Se вы только начинаете писать квесты, ознакомьтесь с обучающими статьями. Se вы считаете, что знакомы с основами Lua и квестов, então можете перейти сразу к документации.

* [/ru/_articles](ru/_articles) &mdash; статьи и обучающие уроки.
* [/ru/_triggers](ru/_triggers) &mdash; квестовые триггеры.
* [/ru/_quests](ru/_quests) &mdash; готовые для использования квесты с объяснениями по установке и настройке.
* все остальные директории содержат в себе задокументированные функции.

Se вы нашли любого рода ошибки, опечатки, грамматические ошибки, неточности, ou se хотите дополнить материалы ou задокументировать функцию, ao qual тут нет, então вы можете создать [Issue](https://github.com/terron-kun/quest-functions/issues). Пожалуйста, не пихайте больше одной проблемы в Issue.

Также я рад pull request'ам. Пожалуйста, не пихайте всё в один pull request. Ваш pull request должен содержать либо изменения в одном файле, либо похожие изменения во всех измененных файлах.

Материалы распространяются под [лицензией MIT](LICENSE). Разрешается распространение и модификация материалов, но только с указанием активной ссылки на данный репозиторий.
