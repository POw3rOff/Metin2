# dance_event.gohome()
Функция **dance_event.gohome** телепортирует всех игроков из локации с индексом `115` в первые города.

## Примечания
Функция может быть вызвана анонимно.

Телепортируются все, кроме администраторов.

Серверные координаты игроков долнжы быть внутри квадрата (764503, 22807), (772362, 26499).

Эта функция просто телепортирует игроков в первый город из локации с индексом `115` и указанными выше координатами. Что такое &laquo;танцевальный эвент&raquo; неизвестно.
