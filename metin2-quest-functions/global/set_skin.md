# set_skin()
A função **set_skin** altera стиль квестового окна.

## Parâmetros da função
### skin_id
Tipo *number*. **Parâmetro obrigatório**. Параметры могут принимать значения от `0` до `3`. `0` &mdash; квестовое окно не будет отображаться; `1` &mdash; квестовое окно будет иметь классический дизайн; `2` &mdash; предположительно, этот скин define жирные черные полосы сверху и снизу экрана, при этом само квестовое окно не отображается; `3` &mdash; квестовое окно будет иметь дизайн свитка.

## Notas
A função **não** pode ser chamada anonimamente.
