# get_locale_base_path()
A função **get_locale_base_path** informa относительный путь от ядра до папки с локализацией.

## Valores de retorno
### path
Tipo *string*. Относительный путь от ядра до папки с локализацией.

## Notas
A função pode ser chamada anonimamente.

Папка с локализацией &mdash; это та, onde лежат файлы дропа, групп монстров и т.п.
