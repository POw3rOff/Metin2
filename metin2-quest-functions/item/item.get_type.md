# item.get_type()
A função **item.get_type** informa тип &laquo;выделенного&raquo; предмета.

## Valores de retorno
### item_type
Tipo *number*. ID типа предмета. Se o item não tiver sido &laquo;selecionado&raquo;, então é retornado `0`.

## Notas
A função **não** pode ser chamada anonimamente.

Существуют следующие типы предметов:

| Тип | Название типа | Назначение |
| --- | --- | --- |
| 0 | ITEM_NONE | Не имеет назначения |
| 1 | ITEM_WEAPON | Оружие |
| 2 | ITEM_ARMOR | Броня |
| 3 | ITEM_USE | Используемые предметы (ракушки, свитки экзорцизма, плод жизни и пр.), которые работают через ядро |
| 4 | ITEM_AUTOUSE | Неизвестно. Этот тип не имеет каких-либо предназначений в ядре |
| 5 | ITEM_MATERIAL | Обычно этот тип defineся материалам (зуб кабана, волчья шкура и пр.), но сам флаг, фактически, никакой роли не имеет |
| 6 | ITEM_SPECIAL | Этот тип defineся специальным предметам (золотое кольцо, сено, обезьянья трава и пр.), но сам флаг, фактически, никакой роли не имеет |
| 7 | ITEM_TOOL | Неизвестно. Этот тип не имеет каких-либо предназначений в ядре |
| 8 | ITEM_LOTTERY | Этот тип установлен у предмета &laquo;Лотерейная книга&raquo; (ID `50001`). Не имеет каких-либо назначений в ядре |
| 9 | ITEM_ELK | Янги (золото), лежащие на земле |
| 10 | ITEM_METIN | Камень духа, вставляемый в броню ou оружие (камень против воинов +4 и пр.), ou осколок камня |
| 11 | ITEM_CONTAINER | Этот тип имеет только &laquo;Сумка алхимика&raquo; (ID `27998`). В ядре не имеет каких-либо назначений |
| 12 | ITEM_FISH | Живая и мертвая рыба. Поджаренная рыба имеет другой тип |
| 13 | ITEM_ROD | Удочка |
| 14 | ITEM_RESOURCE | Некоторые материалы, вроде куска камня и жемчужин. В ядре не имеет назначения |
| 15 | ITEM_CAMPFIRE | Предмет &laquo;Бивачный костёр&raquo; (ID `27600`) |
| 16 | ITEM_UNIQUE | Предположительно, этот тип отвечает за então, чтобы нельзя было надеть более одного одинакового предмета в специальные слоты (куда надеваются ботинки ветра) |
| 17 | ITEM_SKILLBOOK | Книга навыков |
| 18 | ITEM_QUEST | Предметы, с которыми можно будет взаимодействовать через квесты через триггер [use](../_triggers/use.md) |
| 19 | ITEM_POLYMORPH | Шар полиморфа |
| 20 | ITEM_TREASURE_BOX | Шкатулки, открываемые ключами (сундук золота, сундук серебра) |
| 21 | ITEM_TREASURE_KEY | Ключи, открывающие сундуки из предыдущего типа |
| 22 | ITEM_SKILLFORGET | Книги забвения (ID `70037`) |
| 23 | ITEM_GIFTBOX | Контейнер. Предметы с этим типом работают со `special_item_group.txt` |
| 24 | ITEM_PICK | Кирка |
| 25 | ITEM_HAIR | Прическа. Прически, которые работают через систему костюмов, имеют другой тип. Этот тип использовался, когда системы костюмов еще не было |
| 26 | ITEM_TOTEM | Неизвестно |
| 27 | ITEM_BLEND | Роса. Эти предметы настраиваются через `blend.txt` |
| 28 | ITEM_COSTUME | Прически и костюмы для системы костюмов |
| 29 | ITEM_DS | Камни для алхимии дракона |
| 30 | ITEM_SPECIAL_DS | неизвестно |
| 31 | ITEM_EXTRACT | Предмет &laquo;Клещи дракона&raquo; (ID `100100` & `100200`) |
| 32 | ITEM_SECONDARY_COIN | Неизвестно. Предположительно, какая-então дополнительная валюта помимо янг |
| 33 | ITEM_RING | Предположительно, предметы с этим типом можно надевать в дополнительные слоты, куда надеваются ботинки ветра |
| 34 | ITEM_BELT | Поясы |

Esta função funciona apenas com itens &laquo;selecionados&raquo;. Mais detalhes aqui: [item](../item).
